# Head Ball Inspired

A 2D football game inspired by Head Ball.

## Description

This project is a fan-made game that brings the exciting gameplay of Head Ball to your PC.

## Controls

*   **A**: Move Left
*   **D**: Move Right
*   **Space**: Jump
*   **E**: Normal Kick
*   **Q**: High Kick
